# Epay Docker 版

> 本项目基于 [maajiko/Epay](https://github.com/maajiko/Epay)（彩虹易支付系统）转换而来，使用 AI 辅助完成 Docker 化改造，方便一键容器化部署。
> 
> **原作者仓库**：https://github.com/maajiko/Epay

---

## 快速开始

1. 克隆仓库
   ```bash
   git clone https://github.com/yourname/epay-docker.git
   cd epay-docker
   ```

2. 复制环境配置文件并修改数据库信息
   ```bash
   cp .env.example .env
   # 编辑 .env，填写你的数据库地址、用户名和密码
   ```

3. 启动服务
   ```bash
   docker compose up -d
   ```

4. 访问后台
   - 后台地址：`http://localhost:8080/admin/login.php`
   - 默认账号：`admin`
   - 默认密码：`123456`
   - **登录后请立即修改默认密码**

---

## 环境变量说明

| 变量名 | 说明 | 默认值 |
|---|---|---|
| `DB_HOST` | MySQL 数据库地址 | `your_db_host` |
| `DB_PORT` | MySQL 端口 | `3306` |
| `DB_USER` | 数据库用户名 | `your_db_user` |
| `DB_PASSWORD` | 数据库密码 | `your_db_password` |
| `DB_NAME` | 数据库名 | `epay` |
| `DB_PREFIX` | 表前缀 | `pay` |

---

## 技术栈

- **PHP 8.0-FPM** + **Nginx**（同容器架构，简化部署）
- **MySQL 8.0**（支持外接独立数据库，无需 Docker 内建）
- **Composer** 依赖管理
- **Docker & Docker Compose**

---

## 注意事项

- 首次启动时会自动导入 `install/install.sql` 初始化数据库，并生成 `install.lock` 安装保护文件。
- 容器内 PHP 已统一设置为 `UTF-8` 编码，避免中文乱码。
- 生产环境建议配置 HTTPS，并修改默认管理员密码。

# 彩虹易支付系统（原作者说明）

**彩虹易支付系统** 由郑州追梦网络科技有限公司开发，是一款开源的免签约支付产品，能够帮助开发者一站式接入支付宝、微信、财付通、QQ钱包等多种支付方式，实现高效的支付集成。

---

## 功能特色

- **多渠道支付集成**：支持支付宝、微信、财付通、QQ钱包、微信WAP、银联等多种支付方式
- **便捷的支付解决方案**：简化支付流程，支持快速集成和上线，提供完整的 API 接口
- **后台管理和数据统计**：提供支付统计、代付统计、利润分析等多种后台管理功能
- **安全可靠**：采用 RSA 公私钥验证，支持风控检测和黑名单管理
- **插件扩展**：支持丰富的支付插件，可根据需求灵活扩展
- **移动端优化**：全新的手机版支付页面，支持各种移动端支付场景
